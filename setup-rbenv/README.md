## Run install.sh to install rbenv
```
bash install.sh
```
## Install rbenv ruby version 2.6.1
```
rbenv install 2.6.1
```
## use as global version rbenv ruby version 2.6.1
```
rbenv global 2.6.1
```
## Install bundle
```
gem install bundler
```
## Install rails version 6.0.0.rc1
```
gem install rails -v 6.0.0.rc1
```
